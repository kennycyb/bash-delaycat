# bash-delaycat
CAT with delay

```
cat README.md | ./delay
```

# Usage


```
./delay --help
```


```
Usage:

-d=0.1,  --delay=0.1       delay each line with 0.1 seconds
-f=1000, --first=1000      start on line 1000
-l=1000, --line=1000       maximum 1000 lines
-s=10,   --per-seconds=10  delay each line with 0.1 seconds / 10 line per seconds
-m=60,   --per-minutes=60  delay each line with 1 seconds / 60 line per minutes

-h,      --help            display this help and working parameters
```
